# When giving more makes you look worse: paradoxical inferences in a Bayesian model of social evaluation

## Madeleine Horner, Victor Btesh, Tadeg Quillien

People readily infer how much others care about their welfare from prosocial actions. However, these inferences become more difficult when there is uncertainty over the resources someone has to share, a common real-world scenario. We develop a Bayesian computational model of how people make welfare trade-off ratio (WTR) inferences under resource uncertainty. This model predicts that under uncertainty people marginalize, integrating over all possible resources when evaluating donations. In a behavioral study (N = 129), we found that across donation amounts participants' WTR estimates closely tracked those made by our Bayesian model. Notably, participants exhibited the following paradoxical pattern predicted by our model, WTR estimates dropped once the donation amount (\$25) revealed that the actor was rich. These findings demonstrate that people are able to make sophisticated inferences consistent with rational Bayesian reasoning, updating their beliefs about a partner's resources and adjusting their WTR estimates accordingly. 


# Structure of this repository
- `code/`: Contains the implementation of the Bayesian model used to generate predictions.
- `data/`: Contains the raw and processed data from the behavioral study and the behavioral analysis scripts.
- `docs/`: Contains the experiment code used to run the behavioral study.
- `Preregistration/`: Contains the preregistration document for the behavioral study.
